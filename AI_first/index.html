AI_first - Quick Start

Use this page to start the local servers and open the control dashboard on a safe origin (127.0.0.1/localhost).

1) Start both servers (recommended)

Starts the control server (run buttons + live status) and the web server (HTML). Uses .venv/bin/python if present. Stop both with Ctrl+C.

2) Open the dashboard

Always use 127.0.0.1 or localhost so the control API can respond.

Open control dashboard

View-only mode (optional)

If you only need the static HTML (no live status / run buttons), start just the web server:

python3 -m http.server 8000 --directory .
